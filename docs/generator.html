Apex Atlas

Synthetic FHIR generator

Generate synthetic FHIR R4 patients on demand against a running Apex Atlas dev server. Everything here is synthetic — no real patient is depicted.

Requires a running API. Start the dev server locally and point the field below at it: pip install -e ".[dev]" && atlas serve --port 8080. The browser calls it directly (CORS is enabled). This is a development server, not a hosted service.
